# Cadastro de Funcionários (Java)

Projeto simples em Java para cadastro de funcionários, utilizando
`ArrayList`, orientação a objetos e ordenação com `Comparable`.

## Funcionalidades
- Cadastro de funcionários (nome e salário)
- Cálculo de bônus salarial
- Cálculo do salário total com bônus
- Cálculo da média salarial
- Ordenação dos funcionários por salário (do maior para o menor)

## Conceitos Utilizados
- Classes e objetos
- Encapsulamento
- `ArrayList`
- `Scanner`
- Interface `Comparable`
- `Collections.sort()`

## Ordenação
A classe `Funcionario` implementa a interface `Comparable` para permitir
a ordenação automática da lista.

A ordenação é feita pelo salário, do maior para o menor.
